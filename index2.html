Introduction to Probability and Statistics Using R, ISBN: 978-0-557-24979-4, is a textbook written for an undergraduate course in probability and statistics.

IPSUR is FREE, in the GNU sense of the word.

Contact: G. Jay Kerns (gkerns@ysu.edu)

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/gjkerns/IPSUR

get the source code on GitHub : gjkerns/IPSUR
